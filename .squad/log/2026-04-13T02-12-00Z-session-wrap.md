## Key Decisions Made

### PSScriptAnalyzer Requirement: Comments ≠ Statements
Empty catch blocks with only comments violate PSScriptAnalyzer rule PS3109. Must use actual statement (Write-Verbose, Write-Debug, etc.). This applies across all future PowerShell scripts.

### Windows Setup Test Coverage Priority
Comprehensive test suite (15 tests, Groups A-D) now covers:
- Function presence and parameter validation
- Installation idempotency (safe to re-run)
