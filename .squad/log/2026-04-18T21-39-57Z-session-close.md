## Key Learnings

1. **Test-Implementation Sync:** Stale tests create false failures that block CI. When implementation patterns change, test assertions must be updated immediately.

3. **Known Limitations Surface:** Alias injection works partially on some configs. This is a follow-up issue, not a blocker for the current release.
