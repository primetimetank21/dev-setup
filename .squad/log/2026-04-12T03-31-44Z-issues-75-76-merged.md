## Work Summary

### Issue #75 / PR #77 — `feat(setup): add vim to system prerequisites`

- **Root cause:** `install_prerequisites()` in `scripts/linux/setup.sh` was missing `vim` from the `apt-get install -y` list. Aliases `vb` (edit `~/.bashrc`) and `vz` (edit `~/.zshrc`) failed with "command not found" on fresh installs.
- **Fix:** Added `vim` to the `apt-get install -y` line in `install_prerequisites()`.
- **Branch:** `squad/75-add-vim-prerequisite` (deleted after merge)

### Issue #76 / PR #78 — `fix(copilot-cli): use script PTY for non-interactive binary download`

- **Root cause:** `printf 'y\n' | timeout 60 gh copilot` fails in non-TTY environments (e.g. Devcontainer `postCreateCommand`) because `gh copilot` calls `isatty(stdin)` and discards piped input when stdin is not a terminal. Binary was therefore never downloaded.
- **Fix:** Replaced with `printf 'y\n' | timeout 120 script -q /dev/null -c "gh copilot"`. The `script(1)` utility allocates a pseudo-TTY, so `gh copilot` sees `isatty(stdin)=true` and accepts the piped `y`.
- **Branch:** `squad/76-pty-copilot-download` (deleted after merge)
